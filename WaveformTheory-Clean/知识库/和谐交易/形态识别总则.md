# 形态识别总则

> **定位：** 本页回答“如何识别一个谐波形态是否值得继续看”。重点不在死记比率，而在于先看结构骨架，再看比率约束，最后看 PRZ 与确认质量。

## 一、概念定义

谐波形态识别不是“看到五个点就命名”，而是一个分层筛选过程：

1. 先识别 **XABCD 结构骨架** 是否成立；
2. 再检查 **各段回撤与扩展关系** 是否大体符合；
3. 最后确认 **D 点是否把价格带入可信的 PRZ**。

因此，形态识别本质上是“结构 + 比率 + 完成区 + 确认”的联合判断。

## 二、图形理解

### 先看骨架，不先看名字
- 图上首先要能看出清晰的摆动腿，而不是从密集噪音里硬取拐点。
- XA、AB、BC、CD 四段要具备相对明确的方向转换。
- 若骨架本身不顺，后续再精细的比率计算也没有意义。

### 形态识别是筛选，不是拟合
- 真正可用的形态，通常在肉眼上先有“像一个完成结构”的感觉。
- 如果必须靠不断改点位、改级别、改容差才能成立，往往说明结构并不自然。
- 图形越自然，后续量化与执行价值越高。

### D 点只是候选完成，不是自动反转
- D 点进入 PRZ，代表结构到位；
- 是否交易，还要看进入 PRZ 的方式、量价表现、反转确认。

## 三、盘面识别条件

### 必须满足
- 至少存在清晰可辨的 X、A、B、C、D 五个关键点。
- 结构腿之间具有可解释的回撤或扩展关系。
- CD 段指向明确的潜在反转区，而不是无限失控延伸。
- 识别不能脱离所处级别，必须先确定是日线、小时线还是更小级别结构。

### 加分条件
- PRZ 同时与历史支撑阻力、趋势线、缺口、筹码密集区重合。
- D 点附近出现动能衰竭、量价背离、假突破、吞没或反包确认。
- 形态完成位置与威科夫测试/Spring/Upthrust、量价异常形成共振。

### 失效条件
- 拐点靠主观硬拼，换个级别就完全失真。
- 比率关系偏差过大，只能靠放宽条件强行解释。
- 价格穿越 PRZ 后仍顺势强推，没有任何拒绝或停顿。
- 所谓形态只在极小噪音级别成立，无法对应更高一级市场结构。

## 四、常见误判

- 把任何五段波动都当作谐波候选。
- 一看到某个回撤位接近黄金分割，就急于命名形态。
- 只看比率，不看结构是否自然。
- 把到达 D 点当成无条件开仓信号。
- 忽略更大背景，逆着强趋势机械抄顶抄底。

## 五、A股适配说明

- A 股受情绪与涨跌停机制影响，结构拐点常更尖锐，假完成更常见。
- 连板股、消息股、低流动性个股容易制造失真的 XABCD 轮廓，需降低确信度。
- 日线和 60 分钟级别通常比过小周期更有参考价值。

## 六、可量化程度

**中-高。**

可量化方向：
- swing point 提取与级别过滤
- 腿长比率误差
- PRZ 共振项数量
- D 点完成度评分
- 进入 PRZ 后的拒绝强度

## 七、与其他体系的参考关系

- 与 [[量价分析]]：量价决定“这个形态能不能做”，而不只是“像不像”。
- 与 [[威科夫操盘法]]：威科夫负责解释背景与行为，谐波负责给出结构完成区。
- 与 [[艾略特波浪理论]]：波浪可帮助判断该形态处于哪一段结构背景中。
