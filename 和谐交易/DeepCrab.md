# DeepCrab

> **定位：** DeepCrab 可以理解为更深回撤版本的 Crab，核心特征是 B 点更接近 XA 的深回撤，而 D 点仍指向极端扩展完成区。它兼具深修正与末端加速两种特征。

## 一、概念定义

DeepCrab 常见特征为：
- B 点通常回撤 XA 的 **0.886** 附近；
- BC 再修正后，CD 强力扩展；
- D 点通常接近 XA 的 **1.618** 扩展区。

它比普通 Crab 更容易在图上呈现“前段几乎回尽、末端再极端延伸”的节奏。

## 二、图形理解

### 典型图形语言
- B 点深回撤是 DeepCrab 的识别重点。
- BC 常回撤 AB 的 **0.382–0.886**。
- CD 常扩展 BC 的 **2.24–3.618**。
- D 点通常指向 XA 的 **1.618** 扩展区。

### 读图重点
- 要先分清是 DeepCrab 还是 Butterfly，关键在于整体骨架与比率归属。
- AB 与 CD 不只要在价格上体现极端扩展，也要看时间节奏是否仍有结构呼应；若只是靠快速失控推进把 D 点推到外侧，质量通常要下调。
- B 点更深，不代表结构更稳，反而常意味着末端执行更需要确认。
- 若 D 点外侧继续强推，没有拒绝，则失败速度会很快。

## 三、盘面识别条件

### 必须满足
- B 点接近 0.886 是核心识别线索。
- D 点需具备扩展型完成区，而不是普通回撤终点。
- PRZ 需由多项扩展/投影关系共同支撑。

### 加分条件
- D 点与历史极值、趋势通道外侧、情绪高潮位重合。
- 出现长影线、吞没、放量滞涨或放量抗跌等拒绝行为；若只是到位而收盘不能回收，仍不应视为可执行完成。
- 与量价背离、测试失败或末端异动同步。

### 失效条件
- B 点并不深，结构更接近 Crab 或 Butterfly。
- D 点有效穿越扩展位后仍持续顺势延伸。
- 图形只能靠放大缩小不断修正点位才成立。
- 末端若伴随大缺口、异常宽幅或极端收盘，且收盘不能回收，通常更像失控延伸而非结构完成。

## 四、常见误判

- 把所有深回撤 + 极端扩展都叫 DeepCrab。
- 只看 D 点极值，不看 B 点深回撤特征。
- 和 Butterfly 混用。
- 没有确认就因为“位置极端”强行开仓。

## 五、A股适配说明

- A 股中的 DeepCrab 常出现在题材股加速末段或恐慌杀跌尾声。
- 因为 D 点附近经常有过冲和隔夜跳空，所以止损纪律尤其重要。
- 对低流动性标的，这类形态的可解释性会显著下降。

## 六、可量化程度

**中。**

可量化方向：
- B 点 0.886 回撤误差
- D 点相对 XA 1.618 的距离
- 极值过冲幅度
- PRZ 拒绝与回收强度

## 七、与其他体系的参考关系

- 与 [[量价分析]]：可借量价衰竭判断 D 点是不是末端吸收而非继续失控。
- 与 [[威科夫操盘法]]：若 D 点附近对应 Spring / Upthrust 或测试失败，信号更有行为基础。
- 与 [[艾略特波浪理论]]：可细化极端延伸浪段的潜在反转区。
