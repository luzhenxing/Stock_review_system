# 核心结构与比率

> **定位：** 本页建立和谐交易的最小语法：先理解 XABCD 结构，再理解各段之间的回撤与延伸比率关系。

## 一、概念定义

和谐交易的本质不是“记住几个形态名字”，而是识别一组具有内在比例约束的价格摆动结构。最常见表达为：

```text
X → A → B → C → D
```

其中：
- **XA** 是原始推动腿
- **AB** 是对 XA 的回撤
- **BC** 是对 AB 的反向修正
- **CD** 是最后完成段，常把价格带入 PRZ

## 二、图形理解

图形上要先看出“腿”与“拐点”，再去算比率。

### 先看三件事
1. 拐点是否清晰，而不是密集噪音中的随意取点。
2. 结构是否对称或近似对称。
3. CD 是否朝着一个可能的反转区完成，而不是无限延伸。

### 价格比例与时间比例要一起看
- 《斐波那契交易法》的主轴之一，是把价格对称与时间对称放在同一层观察。
- AB=CD 不只看价差是否相近，也要看 AB 与 CD 的时间跨度是否近似对称。
- 若价格比例勉强成立，但时间完全失衡，通常只能降级为低质量候选。

### 图形不是尺子，是骨架
- 比率给的是约束区间，不是机械点位。
- 一张图先看骨架，再看精度；骨架错了，精度越高越危险。

## 三、盘面识别条件

### 必须满足
- 至少能清楚识别 4 个以上关键拐点。
- 各结构腿有相对明确的方向性。
- 回撤/延伸关系不能完全失真。

### 加分条件
- 结构完成区与历史支撑阻力重合。
- 结构完成区与波浪回撤区或趋势通道边界共振。
- 完成段出现量价异常或反转K线确认。

### 失效条件
- D 点过度延伸，破坏关键比率约束。
- B 或 C 点无法形成有意义的回撤结构。
- 图形只在极小级别噪音中成立，放大级别后毫无结构性。

## 四、常见误判

- 把任意五个点都硬凑成 XABCD。
- 只看单一比率，不看整体结构完整性。
- 只因“接近某个黄金分割位”就认定形态成立。
- 忽视 CD 的运行质量与完成方式。

## 五、A股适配说明

- A 股受情绪、涨跌停、板块联动影响，结构可能更“尖锐”或“压缩”。
- 谐波在高波动题材股上更容易出现假完成，需更多量价确认。
- 对日线级别优先于过小周期，避免噪音干扰。

## 六、可量化程度

**中-高。**

可量化方向：
- swing point 提取
- 各腿长度比值
- D 点相对 PRZ 距离
- 结构对称性评分
- 完成区共振评分

## 七、与其他体系的参考关系

- 与 [[艾略特波浪理论]]：都重视 Fibonacci，但谐波更强调局部完成结构。
- 与 [[量价分析]]：量价决定一个结构“能不能做”，而非只看比率“像不像”。
- 与 [[威科夫操盘法]]：若 PRZ 同时处于吸筹末端或 Spring 背景中，质量明显提高。
