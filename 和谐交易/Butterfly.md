# Butterfly

> **定位：** Butterfly 的关键特征是 D 点通常超出 X 点，并完成在 XA 的 1.27 或更深扩展附近。它更像末端延伸后的反转候选，而不是普通回撤结构。

## 一、概念定义

Butterfly 常用于识别“原趋势已经被推到结构极限附近”的形态完成区：
- B 点通常回撤 XA 的 **0.786**；
- CD 最终把价格推到 X 点之外；
- D 点在扩展位上形成更激烈的 PRZ。

因此 Butterfly 常带有更明显的末端情绪与加速特征。

## 二、图形理解

### 典型图形语言
- B 点通常回撤 XA 的 **0.786** 附近。
- BC 常回撤 AB 的 **0.382–0.886**。
- CD 常扩展 BC 的 **1.618–2.618**。
- D 点通常落在 XA 的 **1.27** 或 **1.618** 扩展附近。

### 读图重点
- 先确认 D 点确实越过 X 点，而不是普通 Bat/Gartley。
- Butterfly 的完成常伴随情绪高潮、末端加速或假突破。
- AB 与 CD 不只看价格延展是否够深，也要看时间节奏是否仍保有结构呼应；若只是靠急促失控推进把 D 点推到位，质量要显著下调。
- 若越过 X 点后没有明显拒绝行为，形态失败概率高。

## 三、盘面识别条件

### 必须满足
- B 点接近 0.786 是核心识别条件之一。
- D 点必须突破 X 点外侧，形成扩展型完成。
- PRZ 需要由 XA 扩展与 BC/CD 关系共同支撑。

### 加分条件
- D 点与历史极值、趋势通道边界、情绪高潮位重合。
- 出现长上影/下影、吞没、放量滞涨或放量抗跌等拒绝证据；若只是盘中越过 X 点而收盘不能回收，仍不应视为可执行完成。
- 与 Upthrust、Spring 或量价背离同步。

### 失效条件
- D 点越过 X 点后仍持续顺势放量延伸。
- B 点与 0.786 偏差过大。
- PRZ 过宽，无法形成可管理风险的交易区。
- 越过 X 点后的推进若伴随大缺口、异常宽幅或极端收盘，且收盘不能有效回收，通常更像失控延伸而非结构完成。

## 四、常见误判

- 只要越过 X 点就叫 Butterfly。
- 把任何情绪高潮都解释成末端谐波完成。
- 忽略 B 点与 PRZ 的比例约束。
- 在没有确认的情况下抢反转。

## 五、A股适配说明

- A 股中 Butterfly 常出现在冲板、炸板、加速赶顶或恐慌杀跌末端。
- 因为 X 点外侧过冲很常见，所以更要重视收盘回收而非盘中触达。
- 题材股的 Butterfly 候选容易真假混杂，需结合板块节奏判断。

## 六、可量化程度

**中。**

可量化方向：
- B 点 0.786 回撤误差
- D 点相对 XA 1.27/1.618 的距离
- X 点外侧过冲幅度
- PRZ 进入后回收强度

## 七、与其他体系的参考关系

- 与 [[量价分析]]：Butterfly 的真假高度依赖末端量价是否衰竭。
- 与 [[威科夫操盘法]]：若外侧过冲同时对应 Upthrust 或 Spring，执行意义更强。
- 与 [[艾略特波浪理论]]：常可作为推动浪末端或 C 浪末端的精细执行区。
