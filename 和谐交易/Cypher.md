# Cypher

> **定位：** Cypher 的特征在于 C 点会越过 X 点外侧，而 D 点并不追求极端扩展，而是回到 XC 的关键回撤位附近完成。它是一类“先假突破、后回收完成”的典型结构。

## 一、概念定义

Cypher 与经典 Gartley/Bat/Crab 的区别在于：
- 早段结构更强调外侧穿越；
- C 点常突破 X 点外侧；
- D 点则回到 XC 的关键回撤位附近形成 PRZ。

它常用于识别一次越位后重新被拉回或压回的结构化完成。

## 二、图形理解

### 典型图形语言
- AB 常回撤 XA 的 **0.382–0.618**。
- BC 会扩展 XA 至 **1.13–1.414** 附近，并越过 X 点外侧。
- CD 最终回撤 XC 的 **0.786** 附近完成。

### 读图重点
- Cypher 的灵魂在于“先越位，再回归完成”。
- C 点越出 X 点外侧后，不只要看价格有没有越位，也要看外侧推进与回归段在时间节奏上是否仍保有结构呼应；若只是快速失控冲出后又机械回到位，质量通常要下调。
- C 点越出 X 点外侧后，若只是继续顺势单边推进，则不应强行当 Cypher。
- D 点回到 0.786 附近时，最重要的是有没有拒绝与确认。

## 三、盘面识别条件

### 必须满足
- C 点越过 X 点外侧是关键特征。
- D 点必须靠近 XC 的 0.786 回撤区。
- 结构要能清楚区分 XA、AB、BC、CD 四段。

### 加分条件
- C 点外侧越位同时伴随假突破或情绪脉冲。
- D 点完成时出现量价背离、长影线、吞没或反包；若只是回到 XC 0.786 附近而未见拒绝，仍只算候选完成区。
- 与威科夫假突破、测试或量价背离共振。

### 失效条件
- C 点未越过 X 点外侧。
- D 点未能回到关键 0.786 区附近。
- D 点到位后继续顺势失控延伸。
- 外侧越位或回归段若伴随大缺口、异常宽幅、极端收盘，且收盘不能有效回收，通常更像情绪冲击而非结构完成。

## 四、常见误判

- 只要出现外侧越位就叫 Cypher。
- 不区分 C 点越位与 D 点回归这两个层次。
- 把普通假突破回撤都命名为 Cypher。
- 忽略 D 点确认，只因结构新奇就抢单。

## 五、A股适配说明

- A 股中 Cypher 常见于冲高炸板后回落、或恐慌破位后回收的结构里。
- 因为外侧越位很常由情绪驱动，次日延续与收盘质量尤其关键。
- 对题材股，Cypher 更适合作为确认后的执行参考，而非盘中预测工具。

## 六、可量化程度

**中。**

可量化方向：
- AB 回撤误差
- C 点相对 XA 1.13–1.414 的越位程度
- D 点相对 XC 0.786 的距离
- D 点到位后的回收强度

## 七、与其他体系的参考关系

- 与 [[威科夫操盘法]]：可与 Spring、Upthrust、测试失败等事件形成很强映射。
