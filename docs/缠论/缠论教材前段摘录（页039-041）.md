# 缠论教材前段摘录（页039-041）

## 页039-041 / 上层结构确认、回补分化与原结构边界重估

### 原文明确写出
- 上层结构的成立不能只看局部离开动作本身，还要继续看后续关系是否补齐。
- 离开之后出现回补，并不自动等于原判断失效。
- 有些回补服务于原离开质量确认；有些回补则会把原离开降级，甚至改写原结构理解。
- 结构判断要放在连续演化链里看，不能把单一步骤静态化。

### 图中可见
- 图例不再只画“离开”，而是把离开后的回看、再进入、再离开等后续动作放进同一张结构链里。
- 某些回补虽然重新接近原重叠区，但图上仍保留原方向延续的可能性，没有直接标成彻底失败。
- 另一些回拉则明显吞没原先离开的独立性，使结构重心重新落回旧核心区或转入新的连接框架。
- 图中强调的不是单个点，而是节点之间的先后顺序、连接方式以及回补后是否仍能维持上层结构边界。

### 归纳解释
- 这一段继续把“离开—回看—再延续/失效”细化成更可操作的结构链：不是所有回补都等价，也不是所有重新接近原区间都该一律判死。
- 真正要区分的是三类后果：
  1. **正常质量检验**：回补只是确认原离开是否稳固。
  2. **失败离开**：回补已吞没原离开的独立性，原判断应降级。
  3. **结构重组**：后续连接方式已改变主结构框架，不能再沿用原解释。
- 因此，走势类型与买卖点的确认，不能只靠“曾经离开过”这一瞬间，而要看离开后能否经受回补、再连接与边界重估。
- 这几页对前面知识的推进重点，不是增加新术语，而是把“上层结构确认”从一次动作改成一条连续检验链。

### 触发与失效
- **开始值得观察**
  - 某段结构已出现对原重叠区或原核心区的离开尝试。
  - 后续出现回补、回看、再连接时，应立即进入质量判断，而不是只盯价格强弱。
- **哪一步算确认**
  - 回补后原离开的独立性仍在，且后续连接继续支持原方向或原上层结构。
  - 图上能看出原核心区边界未被重新吞没，或回补后再次展开有效延续。
- **哪一步意味着失效**
  - 回补已把原离开的独立性消解，重新卷入旧重叠核心区。
  - 后续连接关系不再服务于原结构解释，而是迫使主结构重画或改级。
