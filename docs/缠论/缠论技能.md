## 当前建设原则

1. **独立成目录。** 缠论不附着在波浪理论或谐波目录下。
2. **先解释当前字段，再保持与理论层分离。** 当前先说明 `skills.chan` 已落地的 schema，但不让现有字段反向裁剪完整缠论知识。
3. **先做联判角色，再补完理论细节。** 目前的重点是让代码、AI、看板与知识库使用同一套术语。
4. **故事语言不能直接充当 skill 结论。** 如果输出里提到吸货、洗筹、做底、负成本筹码，必须同步落回压力位、假突破/假破位、套牢区脱离、中枢与买卖点这些结构痕迹。
