# PRZ潜在反转区

> **定位：** PRZ（Potential Reversal Zone）是和谐交易最关键的执行概念。它不是一个点，而是一组价格共振形成的反转候选区域。

## 一、概念定义

PRZ 指多个结构比率在同一区域附近汇聚后，形成的“潜在反转区”。

常见构成来源：
- XA 的关键回撤位
- BC 的扩展位
- AB=CD 的投影目标
- 结构边界与历史支撑阻力

## 二、图形理解

### PRZ 不是价格点，而是价格带
- 价格进入 PRZ，只说明“到位了”，不说明“必反转”。
- 真正重要的是：进入方式、量价表现、是否快速拒绝该区域。

### 要看“怎么到达”
- 缓慢衰竭进入 PRZ，常优于加速失控冲进 PRZ。
- 到达 PRZ 前若已出现动能背离或量价失衡，反转概率更高。

## 三、盘面识别条件

### 必须满足
- 至少两个以上关键比率在相近区域共振。
- D 点接近结构完成位置。
- PRZ 具有明确边界，而非极宽泛模糊区间。

### 加分条件
- 与历史高低点、趋势线、缺口、筹码密集区重合。
- 进入 PRZ 后出现缩量衰竭、反包、吞没、假突破等确认。
- 与威科夫测试/Spring/Upthrust 或量价背离同步出现。

### 失效条件
- 价格强势贯穿 PRZ 且无任何拒绝行为。
- 进入 PRZ 后继续放量顺势，不见转弱/转强迹象。
- PRZ 计算基础本身来自失真的结构点。

## 四、常见误判

- 把到达 PRZ 误当成自动开仓信号。
- PRZ 过宽导致任何价格都能被解释为“差不多到了”。
- 忽视进入 PRZ 前的趋势背景与流动性环境。

## 五、A股适配说明

- A 股中 PRZ 常被情绪加速穿刺，尤其在题材股和涨停打开日。
- 因此更需要量价与行为确认，而不是机械挂单。
- 对于流动性不足的个股，PRZ 精度会显著下降。

## 六、可量化程度

**高。**

可量化方向：
- PRZ 上下边界
- 当前价格距 PRZ 的百分比距离
- PRZ 内共振项数量
- 进入 PRZ 后的反向波动幅度

## 七、与其他体系的参考关系

- 与 [[量价分析]]：量价决定 PRZ 的可信度。
- 与 [[威科夫操盘法]]：Spring / 测试 / 上冲失败等行为可做 PRZ 确认。
- 与 [[艾略特波浪理论]]：PRZ 可细化波浪回撤位附近的反转执行点。
