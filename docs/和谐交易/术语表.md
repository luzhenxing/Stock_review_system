# 和谐交易术语表

> **用途：** 统一和谐交易体系中的结构、比率、区域与执行术语，避免后续分析中出现译名混乱。

## 结构与点位
- **X/A/B/C/D**：谐波形态的五个关键转折点。
- **XA**：起始主腿，常作为后续比率测量基准。
- **AB / BC / CD**：结构内的回撤与延伸段。
- **XABCD**：完整谐波结构的标准表达。

## 区域与完成
- **PRZ（Potential Reversal Zone）**：潜在反转区，由多个比率共振形成，不是单一价格点。
- **Completion**：形态完成，通常指价格进入或触及 PRZ。
- **Confluence**：多个比率、结构、支撑阻力的共振。

## 执行与风控
- **Entry Trigger**：进入 PRZ 后的确认触发，不等于一触即买。
- **Invalidation**：结构失效，某个关键比率或点位被破坏。
- **Pattern Failure**：形态完成后未反转，甚至加速反向运行。
- **Measured Move**：以结构长度或关键腿幅进行目标测算。

## 常见形态名
- **Gartley**：经典回撤型谐波结构。
- **Bat**：较深 B 点、较浅 D 点外形的谐波结构。
- **Butterfly**：D 点可能超越 X 点的延伸型结构。
- **Crab / Deep Crab**：以深延伸见长，PRZ 更极端。
- **Cypher / Shark**：非传统 XABCD 变体，强调结构比例特征。

## 使用提示
- 术语必须结合图形理解，不能只记名词。
- PRZ 是“区”，不是“点”。
- 后续若加入更多书中专用译法，在本页统一维护。
