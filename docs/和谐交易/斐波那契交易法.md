# 斐波那契交易法

> **定位：** 本页把《斐波那契交易法》收口为和谐交易可复用的主线知识，只覆盖比例、结构、时间、确认式入场与风险控制，不扩写全书全部基础模型与故事案例。

## 1. 定位
- 该体系解决什么问题：用比例、比率与几何结构把价格波动压缩成可识别的候选模型，并在入场前先量化风险。
- 它更偏结构、执行、节奏、风险，还是确认：同时覆盖结构识别、执行确认与风险优先，但不承担单独终判。
- 本次学习范围：斐波那契数列与黄金分割、AB=CD、常见回撤/扩展、时间对称、Gartley/Butterfly 等核心模型的共性、确认式入场、波动率止损、危险信号与资金管理。
- 本次明确不覆盖：十个基础模型逐个拆页、全书全部图例逐图复刻、所有经典图形变体、交易轶事与心理叙事的独立扩写。

## 2. 核心定义
- **斐波那契比例** 不是神秘点位，而是用来约束价格摆动关系的测量语言，书中主轴反复回到 0.382、0.50、0.618、0.707、0.786、1.27、1.618。
- **AB=CD** 是最核心的结构骨架之一。它不是单独预测工具，而是与回撤位、扩展位、时间段共同组成候选完成区。
- **时间对称** 与价格对称同样重要。书中多次强调，波段长度不仅看价格比例，也看 K 线数量或时间跨度是否近似相称。
- **模型识别** 的目的不是猜顶猜底，而是把风险压缩到可定义区域后等待执行确认。

## 3. 结构单元
- 最小单元：一条主推动腿、一次回撤、一次再展开，以及可测量的完成段。
- 组合方式：先找清晰摆动骨架，再验证回撤/扩展比例，最后检查是否形成 AB=CD 或扩展 AB=CD。
- 阶段推进关系：
  1. 先识别摆动结构；
  2. 再比较价格比例；
  3. 再检查时间对称；
  4. 最后把完成区压缩成可观察的执行候选。
- 书中对 Gartley“222”和 Butterfly 的强调，本质都回落到同一主链：**结构骨架 + 比例共振 + 时间对称 + 完成区确认**。

## 4. 图形理解
- 图片中必须识别的结构：主推动腿、回撤腿、AB=CD 对称、D 点附近的完成方式、是否越过 X 点、是否存在扩展完成。
- OCR 容易漏掉的视觉信息：
  - 两段波动在价格与时间上的呼应；
  - D 点进入目标区时的速度变化；
  - X 点外侧过冲与回收；
  - 图上用来表达“候选完成”而非“立即反转”的位置关系。
- 文字与图形如何互证：正文给出比例范围，图形负责说明这些比例如何长成自然骨架；若图上节奏失真，即使数字接近也不应升格为高质量模型。

## 5. 触发与失效
- 触发条件：价格到达几何完成区后，出现十字星、锤线、流星、镊子、回收、衰竭或其他确认迹象。
- 确认条件：作者明确偏好确认式入场，而不是一触即做；“夏皮罗反复”强调至少等待当前决策周期的一根完整 K 线，以过滤情绪化追单。
- 失效条件：
  - 完成区被持续贯穿且无回收；
  - 结构越过关键 X 点后继续顺势扩展；
  - 价格与时间对称被破坏，只能靠放宽解释硬凑模型。
- 常见假信号：
  - 只命中单一比例就急于命名；
  - 看到 PRZ 或 D 点就机械抄底抄顶；
  - 入场后仍未提前定义止损与目标。

## 6. 与其他体系协同
- 与 wave / harmonic / chan / ruthless / 威科夫 / 量价 的关系：
  - 更直接服务于 harmonic 的比率与完成区语言；
  - 可为 wave 提供回撤结束区与扩展目标的精细化校准；
  - 可为 ruthless 提供更明确的入场、止损与移损节奏；
  - 需由量价与威科夫帮助判断完成区是真衰竭还是趋势延续。
- 它在联判中负责回答什么问题：
  - 当前是否存在结构上可测量的完成区；
  - 风险是否能在结构上被量化；
  - 这次反转候选更像自然完成还是情绪失控后的假到位。

## 7. AI 可用映射
- 可沉淀为 skill 的部分：回撤/扩展比率、AB=CD 候选、Butterfly/Gartley 等模式标签、PRZ 边界、fit score、completion。
- 可沉淀为 snapshot 的部分：当前候选形态、关键比例、PRZ 区间、是否已接近完成。
- 可沉淀为 decision_context 的部分：
  - 是否存在确认式入场信号；
  - 时间对称是否支持当前候选；
  - 是否已出现危险信号，如宽幅、缺口、极端收盘。
- 仅适合作为辅助证据的部分：交易轶事、心理故事、关于市场秩序的叙事性表达。

- 是否阻塞主线：否。

## 9. 后续优化提示
- 对 producer / parser / prompt / UI / backtest 的启发：
