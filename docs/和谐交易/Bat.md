# Bat

> **定位：** Bat 形态的核心特征是 B 点回撤相对较浅，而 D 点通常完成在 XA 的 0.886 附近。它常用于识别更深一层但仍属结构化的修正完成区。

## 一、概念定义

Bat 与 Gartley 同属经典谐波家族，但节奏通常表现为：
- B 点回撤更浅，说明原始推动腿的主导力量尚未被完全抵消；
- D 点完成更深，说明最后一段修正把价格压入更极致的 PRZ。

因此 Bat 常兼具“结构仍完整”与“完成位置更深”的特征。

## 二、图形理解

### 典型图形语言
- B 点通常回撤 XA 的 **0.382–0.50** 区间。
- BC 常回撤 AB 的 **0.382–0.886** 区间。
- CD 常扩展 BC 的 **1.618–2.618** 区间。
- D 点通常接近 XA 的 **0.886** 回撤位。

### 读图重点
- Bat 的早段回撤偏浅，意味着结构骨架通常更顺。
- AB 与 CD 不只要在价格上形成配套，也要看时间跨度是否近似呼应；若只是价位接近，但完成节奏明显失衡，质量通常要下调。
- D 点虽然更深，但仍应表现为“衰竭式完成”，而不是失控式击穿。
- 若 CD 末端太情绪化，容易从 Bat 候选退化为失败结构。

## 三、盘面识别条件

### 必须满足
- B 点不能太深，否则更接近 Gartley 或其他结构。
- D 点附近要有清晰 PRZ，而不是宽泛区域。
- BC 与 CD 要具备可解释的展开关系。

### 加分条件
- PRZ 与前低前高、趋势线、缺口或筹码密集区重合。
- D 点进入 XA 0.886 附近后出现量缩衰竭、背离、吞没或假突破确认；若只是触及而未见拒绝，仍应停留在候选阶段。
- 与大级别趋势修正末端背景一致。

### 失效条件
- B 点过深，破坏 Bat 早段特征。
- D 点有效跌穿/涨穿 0.886 支撑的结构边界且无回收。
- CD 进入 PRZ 后仍顺势强推，或以大缺口、异常宽幅、极端收盘的方式直接把候选完成区打穿。

## 四、常见误判

- 把所有 0.886 附近的回撤都叫 Bat。
- 忽略 B 点相对较浅这一关键差异。
- 不看 BC/CD 节奏，只看 D 点到位。
- 把 Bat 和 Butterfly 混淆。

## 五、A股适配说明

- A 股中 Bat 常出现在强趋势回调末端，但盘中尖刺会让 D 点经常假穿。
- 对高波动题材股，Bat 更需要收盘回收与次日确认。
- 在弱流动性个股上，Bat 的 PRZ 精度会明显下降。

## 六、可量化程度

**中-高。**

可量化方向：
- B 点浅回撤误差
- D 点相对 XA 0.886 的距离
- BC/CD 扩展匹配度
- PRZ 内拒绝与回收强度

## 七、与其他体系的参考关系

- 与 [[量价分析]]：Bat 的可执行性依赖 PRZ 内的量价确认。
- 与 [[威科夫操盘法]]：若 D 点对应测试通过或末端吸收，反转质量更高。
- 与 [[艾略特波浪理论]]：常可用来细化一段修正浪的结束区。
