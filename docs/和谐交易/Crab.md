# Crab

> **定位：** Crab 以极端的 D 点扩展著称，核心是利用 XA 的 1.618 扩展附近寻找末端衰竭反转区。它不是普通修正结构，而是更强调“过度延伸后的极值完成”。

## 一、概念定义

Crab 的典型特征是：
- B 点通常回撤 XA 的 **0.382–0.618**；
- BC 再修正后，CD 最终强力扩展；
- D 点通常落在 XA 的 **1.618** 扩展附近。

因此 Crab 更适合用于识别趋势或情绪被推至极端后的反转候选。

## 二、图形理解

### 典型图形语言
- B 点相对 XA 通常不深。
- BC 常回撤 AB 的 **0.382–0.886**。
- CD 常扩展 BC 的 **2.24–3.618**。
- D 点通常接近 XA 的 **1.618** 扩展位。

### 读图重点
- Crab 的末端往往很尖锐，图上常见加速冲刺。
- AB 与 CD 不只看价格扩展是否够深，也要看时间节奏是否仍有结构呼应；若只是靠急促失控推进把 D 点推到 1.618，质量通常要下调。
- 这类结构不应只因“碰到 1.618”就抢反转，必须等拒绝行为。
- 若 D 点附近完全没有衰竭，只能说明趋势仍在失控延伸。

## 三、盘面识别条件

### 必须满足
- D 点必须具备明显的扩展型特征，而非普通回撤完成。
- XA 1.618 附近需要有可定义 PRZ。
- BC 与 CD 的展开关系不能完全失真。

### 加分条件
- D 点与历史极值、情绪高潮位、通道外侧重合。
- 末端出现放量滞涨、假突破、长影线、吞没或反包；若只是触及 XA 1.618 而未见回收或拒绝，仍只算候选完成区。
- 与量价背离、威科夫 Upthrust / Spring 同步。

### 失效条件
- D 点有效穿越 1.618 后仍持续单边推进。
- CD 末端没有任何停顿、回收或衰竭迹象。
- 拐点与比率关系过度主观拼接。
- 末端若以大缺口、异常宽幅或极端收盘的方式直接穿越 PRZ，通常更像失控延伸而非极值完成。

## 四、常见误判

- 看到 1.618 就认定 Crab 完成。
- 把任何极端加速都解释成 Crab，而不看早段结构。
- 忽略 PRZ 外侧仍可能继续过冲。
- 在无确认时逆势抄顶抄底。

## 五、A股适配说明

- A 股里 Crab 常出现在加速板、情绪赶顶、杀跌宣泄末端。
- 由于涨跌停与尾盘脉冲容易制造极值过冲，收盘位置比盘中触达更重要。
- 对高弹性题材股，Crab 的执行必须更保守。

## 六、可量化程度

**中。**

可量化方向：
- D 点相对 XA 1.618 的误差
- BC/CD 扩展匹配度
- 末端过冲幅度
- PRZ 内回收与拒绝强度

## 七、与其他体系的参考关系

- 与 [[量价分析]]：Crab 的可做性取决于极值区是否出现努力与结果背离。
- 与 [[威科夫操盘法]]：若末端极值同时对应 Spring 或 Upthrust，信号质量更高。
- 与 [[艾略特波浪理论]]：常可细化推动浪末端或 C 浪加速段的完成区。
