# Gartley

> **定位：** Gartley 是最经典的 XABCD 谐波形态之一，核心特征是 B 点回撤较深、D 点通常落在 XA 的 0.786 附近，常用于识别趋势中的修正完成区。

## 一、概念定义

Gartley 强调“趋势中的结构化回撤”：
- XA 是原始推动腿；
- AB 对 XA 进行关键回撤；
- BC 对 AB 再修正；
- CD 最终把价格带入 PRZ。

它更像一类“顺大背景、等局部结构完成”的反转候选，而不是完全独立于趋势的拐点预测工具。

## 二、图形理解

### 典型图形语言
- B 点通常回撤 XA 的 **0.618** 附近。
- BC 常回撤 AB 的 **0.382–0.886** 区间。
- CD 常扩展 BC 的 **1.272–1.618** 区间。
- D 点通常接近 XA 的 **0.786** 回撤位。

### 读图重点
- 先看结构是否自然，而不是先套 0.786。
- Gartley 往往出现在趋势修正末端，图上应能看出“推进—回撤—再修正—完成”。
- AB 与 CD 不只看价格比例，也要看时间跨度是否大致呼应；若价位勉强接近，但节奏明显失衡，通常只能降级为低质量候选。
- D 点若只是机械命中比率，但进入方式失控，则交易价值下降。

## 三、盘面识别条件

### 必须满足
- X、A、B、C、D 五点清晰。
- B 点不能过浅或过深，偏离 0.618 太多时可信度下降。
- D 点必须由多重比率共振支持，而非单一比率。

### 加分条件
- PRZ 与历史支撑阻力、趋势线、缺口或筹码区重合。
- D 点进入 PRZ 后出现量价背离、假突破、吞没或反包确认；仅到位不确认，仍只算候选完成区。
- 与威科夫测试、Spring、Upthrust 或量价衰竭同步。

### 失效条件
- B 点严重偏离关键回撤位。
- D 点有效穿越 PRZ 后仍顺势延伸。
- CD 段完全失控，图形节奏被破坏，或以大缺口、异常宽幅、极端收盘的方式直接穿越候选完成区。

## 四、常见误判

- 只因为 D 点接近 0.786 就认定是 Gartley。
- 把任何五段回撤都命名为 Gartley。
- 忽略 BC 与 CD 的配套关系。
- 在强单边趋势里逆势机械抄底抄顶。

## 五、A股适配说明

- A 股中 Gartley 常被情绪脉冲穿刺 PRZ，需要更重视收盘与次日反馈。
- 在热点题材股上，Gartley 更容易出现“先过冲后回收”的假完成。
- 日线和 60 分钟级别通常比更小周期更有参考价值。

## 六、可量化程度

**中-高。**

可量化方向：
- B 点相对 XA 的回撤误差
- D 点相对 XA 0.786 的距离
- BC/CD 比率匹配度
- PRZ 共振项数量与拒绝强度

## 七、与其他体系的参考关系

- 与 [[量价分析]]：D 点附近若量价不支持，Gartley 只算到位，不算可做。
- 与 [[威科夫操盘法]]：若 PRZ 同时出现测试通过或 Spring/Upthrust，质量更高。
- 与 [[艾略特波浪理论]]：常可映射到趋势中的修正浪完成区。
