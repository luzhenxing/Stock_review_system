# 单针下20战法（补票战法）

> **定位：** 超短线 T+1 补票战法，依托自定义 TDX 指标的"机构线（红线）≥80 + 散户线（白线）≤20"双轨量化条件，捕捉错过第一买点后的二次进场机会。
> 本战法**独立**于 [[艾略特波浪理论]]、[[无情操盘手]]、[[和谐交易]]、[[缠论总览]]、[[道氏理论]]，与它们**平级**，不耦合任何主体系的纪律或买卖点定义；使用时按本目录内规则单独执行。

---

## 核心定义

- **战法性质：** 超短线，**仅持有 1 个交易日**（T+1）
- **应用场景：** 错过第一买点后，依托补票机会再次介入
- **量化条件（双轨）：**
  - **机构线（红线 / 长期）≥ 80** —— 主力筹码持续控盘
  - **散户线（白线 / 短期）≤ 20** —— 短线急跌至超卖
- **图形偏好：** N 型结构连续上升、BBI 趋势向上，**深 V / 双 V / 多 V 完美图形优先**

## 体系定位（与其他体系的关系）

本战法**自成一格**，不向任何主体系的买点系统、买卖点定义或纪律框架反向耦合：

| 维度 | 主体系 | 单针下20战法 |
|------|-------|-------------|
| 时间尺度 | 日线 / 周线趋势 | 超短线 T+1 |
| 信号来源 | 波浪结构 / 均线 / 形态 | 自定义 TDX 双轨指标 |
| 持仓周期 | 趋势级别 | 1 个交易日 |
| 入场逻辑 | B1~B3 / PRZ / 笔买点 | 双轨阈值命中 + 次日开盘介入 |
| 与主仓关系 | 主仓决策依据 | **独立机会，不影响主仓纪律** |

> 使用本战法不改变 [[艾略特波浪理论]] 或 [[无情操盘手]] 的主仓决策；战法之间不混用规则。
