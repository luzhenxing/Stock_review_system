# Claude Code

## Overview

Anthropic 开发的命令行 AI Agent 工具，将 Claude 模型 + Bash 工具 + 文件读写 + 各种内置工具整合在同一终端环境中。支持通过 CLAUDE.md、Rules、Skills 三层机制注入个性化上下文，支持 Subagent 委派和 MCP 工具扩展。是当前 Coding Agent 领域的代表性产品之一。
