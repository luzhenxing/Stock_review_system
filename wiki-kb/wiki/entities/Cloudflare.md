# Cloudflare

## Overview

边缘计算和 CDN 巨头，在 Agent 基础设施领域推出了 Sandboxes（开放完整 OS 访问）、Project Think（下一代 Agents SDK）、Browser Run（Agent 专用浏览器）三大产品。核心创新是 Execution Ladder 分层沙盒模型和 Durable Objects hibernation 机制，让 Agent 空闲时成本趋零、唤醒时毫秒级恢复。
