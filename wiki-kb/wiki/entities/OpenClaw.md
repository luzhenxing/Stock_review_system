# OpenClaw

## Overview

本地持久化 Agent 守护进程，通过心跳机制定期检查任务列表并自主执行。支持 WhatsApp、Telegram、Slack 等多种消息渠道接入，外部事件可以触发 Agent 行动。核心解决的问题：如何让 Agent 在人不在的时候也能持续工作。
