# XinGPT

X（原 Twitter）博主（@xingpt），Agent 化工作流的深度实践者和方法论传播者。以投研和内容生产两大业务的 Agent 化改造为案例，系统输出了个人业务 Agent 升级的完整框架。

## 核心贡献

- 提出**算法杠杆**概念：Agent 化是继时间、产品、平台之后的第四种商业杠杆
- 总结个人 Agent 系统的**三层架构**：知识库 → Skills → CRON
- 将投资决策逻辑拆解为 4 个可执行的 Skill 模板（价值投资/比特币抄底/市场情绪/宏观流动性）
- 系统化研究爆款内容规律，建立可喂给 AI 的内容框架库
- 提出 **AaaS**（Agent as a Service）范式：卖结果而非卖工具
