# HappyClaw

## Overview

开源项目，将 Mac mini 改造成 24 小时在线的 Agent 服务器。该项目本身的代码由 Agent 完成，零手写代码。支持在手机上给 Agent 分配任务，Agent 自主规划执行并返回结果。开发过程中积累的可复用经验可以反哺回 Skill 体系。
