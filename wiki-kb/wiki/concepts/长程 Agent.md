# 长程 Agent

## Definition

Long Horizon Agents——Agent 的终极形态。Agent 不再局限于"即时响应"模式，而是接受更大的目标后自主规划执行步骤，运行数小时甚至数天，只在关键决策点才找人确认。

## Key points

- 两个基础设施缺一不可：记忆机制（Agent 必须记住自己做到哪了）和文件系统权限（能读写文件管理 Context）
- Claude Code 天然满足文件系统权限要求，Rules + Skills 体系解决记忆持久化问题
- HappyClaw 和 OpenClaw 是这个方向的先行者——24 小时在线、手机端发任务、Agent 自主执行
- LangChain 创始人：经过长时间磨合、内化了特定任务模式与背景记忆的 Agent，将形成真正的护城河
- 传统工具没有积累（用了十年的锤子还是锤子），有进化的 Agent 越用越懂你
