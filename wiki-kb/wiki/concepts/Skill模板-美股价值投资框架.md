# Skill模板：美股价值投资框架

XInGPT 在 [[个人业务Agent升级指南]] 中给出的第一个 Skill 模板示例。

```
输入: 公司财报数据
判断标准:
  - ROE > 15%（持续 3 年以上）
  - 负债率 < 50%
  - 自由现金流 > 净利润的 80%
  - 护城河评估（品牌/网络效应/成本优势）
输出: 投资评级（A/B/C/D）+ 理由
```

## 结构解析

- 四个条件均为可量化的标准，没有模糊判断
- 时间维度（持续 3 年）防止单季异常值干扰
- 护城河评估是唯一的定性维度，但限定了三种明确类型
- 输出落到 A/B/C/D 四级，决策清晰

## 扩展思路

XInGPT 声明这只是示例，不代表实际投资标准。真实使用时应根据个人判断标准调整各条件的阈值和权重。
