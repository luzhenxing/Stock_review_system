# Skills 显性化

XInGPT 在 [[个人业务Agent升级指南]] 中强调的核心方法论：**把隐性的判断标准显性化、结构化，让 AI 能按照你的思维框架工作**。

## 为什么需要显性化

大部分人用 AI 的方式是：打开 ChatGPT → 输入问题 → 得到答案。这种方式的根本问题是：AI 不知道你的判断标准。

显性化解决的是：把你脑子里的经验、直觉、判断逻辑，拆解成 AI 可以执行的规则。

## 结构模板

```
Skill 名称: [领域] [用途]
输入: 需要什么数据
判断标准:
  - 条件 1: 可量化标准
  - 条件 2: 可量化标准
  - ...
触发条件:
  - 满足 N 个以上条件 → 信号 A
  - 满足 M 个以上条件 → 信号 B
输出: 评级 + 建议动作
```

## 与 Skill 可组合性的关系

- Skills 显性化解决"把判断写成规则"
- [[Skill 可组合性]] 解决"让多个规则协同工作"
- 两者的结合才能让 Agent 处理复杂决策场景
