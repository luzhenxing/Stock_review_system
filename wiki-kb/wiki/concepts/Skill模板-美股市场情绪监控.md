# Skill模板：美股市场情绪监控

XInGPT 在 [[个人业务Agent升级指南]] 中给出的多维度情绪监控 Skill，用于识别市场顶部风险。

```
监控指标:
  - NAAIM暴露指数: 活跃投资经理的股票持仓比例
    · 数值 > 80 且中位数触及 100 → 机构加仓空间见顶预警
  - 机构股票配置比例: State Street 等大型托管机构数据
    · 处于 2007 年以来历史极值 → 反向预警信号
  - 散户净买入额: 摩根大通追踪的每日散户资金流向
    · 日均买入量 > 85% 历史水平 → 情绪过热信号
  - 标普500远期市盈率: 监控是否接近历史估值峰值
    · 接近 2000 年或 2021 年水平 → 基本面与股价背离
  - 对冲基金杠杆率: 高杠杆环境下的拥挤仓位
    · 杠杆率处于历史高位 → 潜在波动放大器

触发条件:
  - 3 个以上指标同时预警 → 减仓信号
  - 5 个指标全部预警 → 大幅减仓或对冲

输出: 情绪评级（极度贪婪/贪婪/中性/恐慌）+ 仓位建议
```

## 设计思路

- 覆盖机构、散户、估值、杠杆四个维度的交叉验证
- 利用极端值而非常规波动触发信号，减少噪音
- 与 [[Skill模板-宏观流动性监控]] 互补：情绪监控看"贵不贵"，流动性监控看"钱够不够"
