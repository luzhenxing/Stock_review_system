# 个性化 Agent 上下文

## Definition

个人超级 Agent = 通用 Agent + 个性化上下文。通用 Agent 是底座模型 + 工具调用能力（你改不了），个性化上下文是你的偏好、工作流、项目结构、工具链、踩坑记录、经验规律（完全由你定义）。上下文越丰富精准，Agent 越接近一个"懂你"的协作者。

## Key points

- 大部分 RAG/ReAct 自建 Agent 框架没必要做——通用 Agent 的底层能力已经很强大，把时间花在整理业务上下文上回报更高
- 上下文不会过时——模型在变强，但你积累的上下文是平台无关的纯文本，换了更强的模型依然有效
- 上下文具有复利效应：投入的每一分钟都在为未来的自己节省时间
