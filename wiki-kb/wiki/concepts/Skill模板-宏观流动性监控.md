# Skill模板：宏观流动性监控

XInGPT 在 [[个人业务Agent升级指南]] 中给出的流动性监控 Skill 模板。

```
监控指标:
  - 净流动性 = 美联储总资产 - TGA - ON RRP
  - SOFR（隔夜融资利率）
  - MOVE指数（美债波动率）
  - USDJPY + US2Y-JP2Y 利差

触发条件:
  - 净流动性单周下降 > 5% → 预警
  - SOFR 突破 5.5% → 减仓信号
  - MOVE指数 > 130 → 风险资产止损
```

## 实战验证

文章提到了一个真实案例：2026 年 2 月初市场暴跌前 48 小时，这套系统检测到：

- 日债收益率跳涨，US2Y-JP2Y 利差大幅收窄
- TGA 账户余额高企 → 财政部持续从市场抽水
- CME 连续 6 次提高金银期货保证金

Agent 自动匹配历史模式（2022 年 8 月日元套利交易平仓），给出"流动性紧张 + 估值高企 → 减仓"建议，帮助避免了至少 30% 的回撤。

## 关键洞察

- 流动性是市场的血液，比任何单一个股指标更能预示系统性风险
- 净流动性公式（Fed 资产 - TGA - ON RRP）是作者自己总结的核心跟踪指标
- 与 [[Skill模板-美股市场情绪监控]] 组合使用：流动性和情绪同时恶化 = 最高警戒
