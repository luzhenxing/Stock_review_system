# 渐进式披露

## Definition

Progressive Disclosure——Claude Code Skills 的加载策略。Agent 不把几十个 Skill 的全部内容塞进上下文，而是分三层按需加载，避免占满上下文窗口。

## Key points

- **L1**：name + description（约 100 tokens），始终在上下文。Agent 通过 description 决定是否触发
- **L2**：SKILL.md 正文，只在 Skill 被触发时加载
- **L3**：scripts/、references/ 等资源文件，只在正文指引时按需读取
- 几十个 Skill 的 L1 总共几千 tokens，对窗口压力很小
- 几百行的详细操作指南和脚本，只在真正需要时才会被读取
