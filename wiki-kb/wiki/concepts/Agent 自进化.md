# Agent 自进化

## Definition

Agent 通过和人的交互，持续优化自己的决策模式和执行路径。现阶段的"自进化"不是完全自主的——它仍然需要人的反馈来驱动。但关键在于每次反馈都被持久化到文件系统，跨会话有效，不会因为上下文窗口清空而丢失。

## Key points

- **负反馈闭环**：执行任务→出错→用户纠正→写入 rules/ 或更新 Skill→下次自动加载→避免重犯
- **正反馈闭环**：执行任务→成功→用户认可→提炼为新 Skill→能力池扩展→更多任务可委派
- 在 CLAUDE.md 中写入"收到任何纠正后，立即编写规则防止同类错误再次发生"，即可建立自动负反馈机制
- Agent 不是每次都从零开始——它在你积累的所有经验的基础上运行
- 自进化的效果是累积的：用得越久，犯错率越低，能自主完成的任务越多
