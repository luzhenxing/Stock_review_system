# AaaS (Agent as a Service)

XInGPT 在 [[个人业务Agent升级指南]] 中提出的未来商业模式范式，对比传统 SaaS（Software as a Service）。

## SaaS vs AaaS

| 维度 | SaaS | AaaS |
|------|------|------|
| 交付物 | 一个工具 | 一个 Agent |
| 用户行为 | 学习如何使用 | 下达指令 |
| 操作方式 | 自己操作、自己维护 | Agent 自动执行、自动优化 |
| 核心价值 | 卖"能力" | 卖"结果" |

## 核心理念

> 好的 Agent 应该做成服务去替代 SaaS。未来用户不再需要安装软件，而是直接调用 Agent 获得结果。

这与 [[OpenClaw]] 创造者 Peter 的预判一致：未来将是 Agent 的天下。

## 商业路径

从个人 Agent 化到产品化：

1. **自用阶段**：搭建自己的 Agent 系统，跑通闭环
2. **咨询阶段**：帮助同行搭建（如文章中的基金经理案例）
3. **产品化阶段**：做成开源项目（基础免费）+ 高级功能付费订阅或按量计费
